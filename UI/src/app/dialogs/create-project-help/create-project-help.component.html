Starting a new project

A new project can be started on this page.

Fill out the information on each step and when you're done, click Save.

Following is an explanation on each step:

Project Info

Basic information pertaining to the project.

Target audience Info

Information about the target audience to which the project will be presented

Educator's Perception

The educator's view on the target audience, eg.: how they perform, how motivated they are

Learning Objectives

Objectives to achieve when the project is complete.

Affective Objectives (optional)

Affective objectives to achieve when the project is complete.

Social Objectives (optional)

Social objectives to achieve when the project is complete.

Project settings

Whether you'd like to share your project with others or not.

The project status when editing a project.

Invite other users to collaborate on the project.
