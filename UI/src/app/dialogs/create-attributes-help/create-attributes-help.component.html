Creating new attributes

New attributes can be specified on this page.

An attribute is an option that will be available during the process of starting a new Project or Technique.

Definitions can be provided for any of the following sections:

Delivery Mode:

How the subject will be presented to the students.

Interaction:

What type of interaction is expected to be had with the students.

Resolution Scope:

The scope of the desired results.

Social Objective:

Social objectives to be achieved.

Affective Objective:

Affective Objectives to be achieved.

Behaviour:

Behaviour expected when developing a learning objetive.

Task Type:

Type of task to carry out.

During the creation process attributes can be added or deleted.

When you're done, click Save and the currently specified attributes will be saved.
