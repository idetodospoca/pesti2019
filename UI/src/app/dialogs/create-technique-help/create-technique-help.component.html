Starting a new technique

A new technique can be started on this page.

Fill out the information on each step and when you're done, the Save button will become available and the technique can be saved.

Following is an explanation on each step:

Technique Info

Basic information pertaining to the technique, as well as rules to be followed and target audience.

Technique Specifications

The types of delivery mode, resolution scope, interaction and feedback the technique can be applied to.

Educator's Perception

The educator's view on the target audience, eg.: how they perform, how motivated they are.

Learning Objectives

Objectives to achieve when the technique is performed.

Affective Objectives (optional)

Affective objectives to achieve when the technique is performed.

Social Objectives (optional)

Social objectives to achieve when the technique is performed.

Technique Structure

How the technique is to be carried out.

Composed of various modules, each one has one or more phases, with each phase carrying out a specific set of tasks.

New modules, phases and tasks can be added or deleted to best describe the technique.
